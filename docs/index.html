About

This site hosts interactive data visualisations produced as part of the Data Visualisation and Data-driven Decision Making exam at the IT University of Copenhagen.

All visualisations are built with open-source libraries — D3.js and Vega-Lite — and run entirely in the browser with no backend required.

Tech stack

  • D3.js v7 Custom interactive charts
  • Vega-Lite v5 Declarative visualisations
  • Vanilla JS No build step needed
  • GitHub Pages Free public hosting
