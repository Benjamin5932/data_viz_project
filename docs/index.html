The Invisible Fleet

Visible fishing versus hidden activity in the North Atlantic, 2017–2019

Not all fishing leaves a trace. When vessels turn off their tracking signals, they become invisible to global monitoring systems. This investigation reveals where, when, and by whom.

About This Investigation

This analysis examines fishing vessel tracking data from 2017–2019 to reveal patterns in "invisible" fishing activity—periods when vessels disable their Automatic Identification System (AIS) tracking.

Data comes from Global Fishing Watch, which combines satellite AIS data, vessel registries, and machine learning to monitor industrial fishing.

This site was built for the Data Visualisation and Data-driven Decision Making exam at the IT University of Copenhagen, 2026.

Methodology

  • Gap Detection: Periods where tracked vessels suddenly go offline
  • Spatial Aggregation: 1° × 1° grid cells for regional clustering
  • Temporal Scope: Monthly summaries across 2017–2019
  • Gear Classification: Vessels grouped by fishing method
  • Flag Analysis: Vessels grouped by flag state of registry
